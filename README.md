# 📝 Note App

A fully **responsive note-taking application** built with modern web technologies. Users can create, organize, and easily find their notes with filtering options.

## 🚀 Technologies Used

* React
* Tailwind CSS
* react-redux
* @reduxjs/toolkit
* redux-persist
* react-markdown
* react-simplemde-editor
* EasyMDE
* uuid
* react-router-dom

## 📌 Features

* 🆕 **Create New Notes:** Add notes with a title, description, and tags.
* 📋 **Notes Listing:** View all created notes in a structured list.
* 🔍 **Filter by Title:** Quickly search notes by their title.
* 🏷️ **Filter by Tag:** Easily find notes by tags.
* ✍️ **Markdown Support:** Write and preview notes in markdown format.
* 💾 **Persistent Storage:** Notes are saved with redux-persist and remain after page refresh.
* 📱 **Responsive Design:** Optimized for mobile, tablet, and desktop devices.
